Hệ thống nhận diện vật thể nhúng
trên nền tảng ESP32-S3

Sơ đồ khối hệ thống sử dụng vi xử lý ESP32-S3 kết hợp module camera OV2640, thực hiện thu thập ảnh, xử lý ảnh và nhận diện vật thể trên chip, truyền kết quả qua WiFi.

🔧 Sơ đồ khối phần cứng (Hardware Block Diagram)
📷

Module Camera OV2640

Khối thu thập ảnh
  • CMOS 2MP (1600×1200)
  • Output: JPEG / YUV422 / RGB565
  • Giao tiếp: DVP 8-bit parallel
  • Cấu hình: SCCB (I²C-like)
  • AVDD: 2.5–3.0V, DVDD: 1.3V
🧠

ESP32-S3-WROOM-1

Vi xử lý trung tâm
  • Xtensa LX7 Dual-core @ 240MHz
  • 512KB SRAM + 8MB PSRAM (OSPI)
  • 16MB Flash (Quad SPI)
  • WiFi 802.11 b/g/n + BLE 5.0
  • LCD_CAM peripheral (DVP native)
  • 45 GPIO linh hoạt
⚡

Khối nguồn (Power Supply)

Cấp nguồn hệ thống
  • Input: USB 5V (Type-C / Micro-B)
  • AMS1117-3.3 → 3.3V / 1A (MCU + IO)
  • ME6211 → 1.3V (DVDD OV2640)
  • Tụ lọc: 10µF + 100nF decoupling
  • LED chỉ thị nguồn
DVP 8-bit + SCCB (I²C) + XCLK
🖼️

Tiền xử lý ảnh

Image Preprocessing
  • RGB → Grayscale conversion
  • Gaussian Blur (giảm nhiễu)
  • Adaptive Threshold (tách nền)
  • Morphology (Close + Open)
  • Contour detection (vật thể chính)
🔬

Trích xuất đặc trưng

Feature Extraction
  • HSV Color Histogram (24 bins)
  • LBP Texture (26 bins)
  • Hu Moments - Shape (7 values)
  • GLCM Haralick (6 values)
  • Tổng: 63 đặc trưng / frame
🤖

Phân loại & nhận diện

Classification
  • Nearest Centroid / KNN (k=3)
  • Euclidean Distance matching
  • Database lưu trên Flash (SPIFFS)
  • Kết quả: tên vật thể + confidence
Kết quả nhận diện + Ảnh
📡

Truyền thông WiFi

Wireless Communication
  • WiFi 802.11 b/g/n (2.4GHz)
  • HTTP Web Server (hiển thị kết quả)
  • WebSocket stream ảnh real-time
  • MQTT publish kết quả (IoT)
  • Antenna PCB tích hợp hoặc U.FL
🎛️

Giao diện người dùng

User Interface (on-board)
  • Nút nhấn: Chụp / Mode / Reset
  • LED RGB: trạng thái hệ thống
  • OLED 0.96" I²C (tùy chọn)
  • Buzzer báo kết quả (tùy chọn)
🖥️

Web Dashboard

Remote Monitoring
  • HTML/CSS/JS host trên ESP32-S3
  • Hiển thị ảnh camera real-time
  • Hiển thị kết quả nhận diện
  • Nút điều khiển từ xa
  • Truy cập qua IP local
⚙️ Luồng xử lý dữ liệu (Data Processing Pipeline)

Camera Capture
OV2640 → DVP → PSRAM buffer (QVGA 320×240)
→
STEP 2
Preprocessing
RGB→Gray, Blur, Adaptive Threshold, Morphology
→
STEP 3
Segmentation
Contour detection, Mask generation, Object isolation
→
STEP 4
Feature Extract
HSV(24) + LBP(26) + Hu(7) + GLCM(6) = 63 features
→
STEP 5
Classification
KNN / Euclidean distance → Label + Confidence
→
STEP 6
Output
WiFi → Web Dashboard + OLED display + LED

OV2640 Camera Sensor

OmniVision — Cảm biến ảnh CMOS 2MP
Cảm biến ảnh 1/4" 2MP trên công nghệ OmniPixel2™. Tích hợp bộ mã hóa JPEG trong chip, hỗ trợ đa dạng format output (Raw, YUV, RGB). Cấu hình qua giao thức SCCB.
